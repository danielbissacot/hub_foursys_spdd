## 🏛️ Padrões de Excelência Base

Para ser aprovado pelo Hub, o código Angular deve seguir estes 4 mandamentos:

1.  **Signals First**: Reatividade nativa com `signal`, `computed` e `effect`.
2.  **Standalone Universe**: 100% de componentes e pipes sem NgModules.
3.  **OnPush Detection**: Performance máxima garantida pela estratégia de detecção de mudanças.
4.  **Modern Control Flow**: Uso exclusivo das novas diretivas de controle (`@if`, `@for`, `@switch`).
