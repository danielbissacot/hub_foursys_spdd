# 🧪 Skill: Angular Component Architect

Criação e refatoração de componentes modernos seguindo os padrões de elite das versões v18 e v19+. Foco total em reatividade granular, performance e acessibilidade.

### 🤖 Como acionar no Chat da IA:

### 📚 O que esta Skill entrega:
- **Signals Management**: Uso nativo de `input()`, `output()` e `model()`.
- **Standalone Universe**: Componentes isolados e independentes.
- **OnPush Detection**: Estratégia de detecção de mudanças de alta performance.
- **WACG Compliance**: Componentes acessíveis por padrão.
