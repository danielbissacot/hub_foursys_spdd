# 🛠️ Skills de Spring Boot (Habilidades de Execução)

Nesta pasta ficam os roteiros de **Ação** da IA. Enquanto as `Instructions` dizem *o que é certo*, as `Skills` dizem *como fazer*.

### 🧩 Como funcionam as Skills?
Cada skill é um arquivo markdown que contém um fluxo de trabalho detalhado para uma tarefa específica.

### 📚 Exemplos de Skills para Criar:
- `gerar-massa-swagger.md`: Como criar JSONs de exemplo baseados no `@RequestBody`.
- `validar-checkstyle.md`: Como identificar e corrigir erros de estilo automaticamente.
- `criar-mappers-mapstruct.md`: Como gerar mappers respeitando a Arquitetura Hexagonal.
- `refatorar-records.md`: Como converter classes DTO para Records do Java 21.

---
> **Uso pelo Agente:** O Mentor (`mentor-spring-hexagonal.md`) tem permissão para consultar esses arquivos e seguir seus passos para executar tarefas repetitivas com perfeição.
