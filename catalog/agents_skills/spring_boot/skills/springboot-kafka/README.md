# ⚙️ Skill: Spring Boot Kafka

Implementação robusta de mensageria assíncrona e arquitetura orientada a eventos (EDA) utilizando Kafka e Confluent Cloud.

### 🤖 Como acionar no Chat da IA:

### 📚 O que esta Skill entrega:
- **Padrões de Produção**: Envio assíncrono com headers e chaves de particionamento.
- **Resiliência Nativa**: Estratégias de Retry e Dead-Letter Topics (DLT).
- **Arquitetura Hexagonal**: Isolação total entre Kafka e Domínio.
