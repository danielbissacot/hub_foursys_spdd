# 🧭 Pilar de Instruções (Governança Central)

Nesta pasta estão as "leis" de desenvolvimento da nossa empresa. São as instruções globais que todas as IAs devem seguir para garantir a qualidade do código.

### 🏛️ Os 6 Pilares:
1. **Java Hexagonal/MVC**: Padrão de pastas e fluxos.
2. **Angular v18+**: Tecnologias front-end atualizadas.
3. **Segurança e LGPD**: Proteção de dados e OWASP.
4. **SOLID e Clean Code**: Manutenibilidade do código.
5. **Testes e BDD**: Qualidade garantida com comportamento.
6. **Docker e DevOps**: Padrões de infraestrutura.

---
> **Como usar:** No chat da IA, dê o comando: `@instructions` para carregar todas as regras antes de pedir a criação de um novo serviço.
