# Playbooks SDD — Referência e Documentação

> ⚠️ **Atenção**: Os arquivos desta pasta são para leitura humana / referência.
> O `catalog-loader.ts` NÃO lê esta pasta — ele procura em `catalog/sdd/`.

## Para que os playbooks customizados funcionem na extensão

Coloque o arquivo em: `catalog/sdd/generic/foursys-[fase].md`

Exemplo:
- Customizar tasks → `catalog/sdd/generic/foursys-tasks.md`
- Customizar constitution Spring Boot → `catalog/sdd/spring_boot/foursys-constitution.md`
