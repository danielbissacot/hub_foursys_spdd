# 🏗️ Esteira de Prompts (Templates de Desenvolvimento)

Nesta pasta estão as 5 Fases padronizadas do ciclo de vida da nossa entrega, garantindo consistência desde a história do usuário até o código final.

### 📜 As 5 Fases:
- **1_prompts_historias**: Transformação dos requisitos de negócio em Historias de Usuário ricas.
- **2_prompts_regras_tecnicas**: Definição da stack tecnológica e design de arquitetura.
- **3_prompts_validadores**: Code Review, Validação Hexagonal e Testes Unitários.
- **4_prompts_entrega**: Documentação final, notas de release e padrão de commit.
- **5_templates_cobol**: Padronização, Modernização e manutenção de sistemas Mainframe.

---
> **Como usar:** No Chat da IDE, use o `@` e selecione o prompt correspondente à fase de desenvolvimento em que você está.
