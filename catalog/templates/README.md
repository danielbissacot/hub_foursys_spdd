# 📚 Catálogo do Hub de Governança de IA

A pasta `/catalog` contém todos os artefatos de IA da nossa equipe de engenharia. Aqui você encontra as definições, comportamentos e templates a serem usados com as Inteligências Artificiais.

### 📂 Estrutura

- **[`/agents`](./agents)**: System prompts completos para configurar a persona da sua IA (Ex: Tech Lead, Revisor, Arquiteto).
- **[`/instructions`](./instructions)**: Regras corporativas, padrões de codificação ou convenções estritas do time para passar como contexto.
- **[`/templates`](./templates)**: Modelos que agrupam casos de uso.
  - **[`/templates/prompts`](./templates/prompts)**: Repositório de códigos de prompt prontos (famosos "copia e cola"). Seu principal material de uso diário.
