💻 Como usar no VS Code

#

Gatilho Hashtag

Digite # no chat para ver a lista de agentes e regras do Hub.

Contexto

Seleção de Regras

Selecione #HEXAGONAL_JAVA.md para que a IA aplique esse padrão.

🎯

Resultado Focado

A IA gera o código respeitando todas as instruções de segurança e qualidade.

📜 A "Constituição" da IA (instructions)

Regras globais que garantem uma fundação técnica sólida.

🛡️ Segurança

Zero vazamento de dados corporativos.

💎 Qualidade

Clean Code e SOLID por padrão.

🛠️ Cartuchos de Especialistas (skills)

Combine a intenção do seu negócio com as regras do Hub para resultados perfeitos.

📂 Mapa de Ativos

Sabe quando você digita # no VS Code? A lista que aparece vem destes diretórios organizados no Hub. Conheça as responsabilidades de cada um:

Diretório Responsabilidade dos Arquivos
catalog/instructions A "Constituição". Define regras globais de segurança, arquitetura e qualidade.
catalog/agents_skills Os "Especialistas". Instruções focadas em tecnologias específicas (Java, Angular, etc).
catalog/templates A "Planta Baixa". Modelos oficiais para histórias de usuário e estruturas de código.
