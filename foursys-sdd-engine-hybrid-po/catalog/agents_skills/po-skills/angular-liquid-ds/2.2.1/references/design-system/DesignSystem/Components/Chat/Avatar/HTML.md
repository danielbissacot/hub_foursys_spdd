# ChatAvatar

Imagem que pode representar uma pessoa, a BIA ou um assistente do chat.

# Uso do HTML
```
<div class="brad-chat-avatar brad-chat-avatar--medium">
  <em class="i icon-brand-bradesco"></em>
</div>
Copy
```
## Exemplos

Obs: Use o botão Show code abaixo do exemplo para ver o HTML.

Para o uso do avatar é importante seguir a logica de implementação do código, inserir todas as tags na ordem conforme o exemplo.
