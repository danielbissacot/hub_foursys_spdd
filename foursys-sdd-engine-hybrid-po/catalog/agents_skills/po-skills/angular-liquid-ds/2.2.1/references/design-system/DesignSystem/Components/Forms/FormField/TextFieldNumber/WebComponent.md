# TextFieldNumber
## Atualização importante sobre o TextField

Com a criação do web component text-field, foi identificada a oportunidade de unificar o text-field-number dentro deste novo web component.




A partir desta versão, todo o suporte a entradas numéricas está disponível diretamente no text-field, utilizando o atributo type="number" no input.




Essa mudança visa simplificar a utilização e manutenção dos campos de formulário, mantendo uma interface consistente para todos os tipos de entrada.
