# Logos

Logos disponibilizados para uso conforme os padrões do Liquid.

# Instuções de uso:


Use a classe brad-logo, seguida da classe de tamanho, que pode ser brad-size-sm ou brad-size-lg, juntamente com o nome da classe do logo como por exemplo brad-logo__bradesco-institucional-horizontal-default.
A estrutura da classe do logo:
## brad-logo__[nome-do-logo]

# Tamanhos

| Classes | Width |
| --- | --- |
| brad-size-sm | 280px |
| brad-size-lg | 600px |

```
<!-- O exemplo abaixo é do logo "bradesco-institucional-horizontal-default" -->
<img class="brad-logo brad-size-sm brad-logo__bradesco-institucional-horizontal-default" />
Copy
```
