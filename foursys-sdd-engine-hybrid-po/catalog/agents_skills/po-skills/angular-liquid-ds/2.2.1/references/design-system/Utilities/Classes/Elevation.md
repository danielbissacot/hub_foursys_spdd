# Elevation

A propriedade box-shadow permite que você adicione uma sombra em quase qualquer elemento.

## Como determinar elevation (para baixo)

| Classes | Exemplo |
| --- | --- |
| brad-shadow-0 | 0 |
| brad-shadow-10 | 10 |
| brad-shadow-20 | 20 |
| brad-shadow-30 | 30 |
| brad-shadow-40 | 40 |

```
<div class="brad-theme-classic">
  <section class="brad-m-lg-b">
    <div class="examples">
      <div class="brad-card brad-p-md brad-shadow-0">brad-shadow-0</div>
    </div>
  </section>
</div>
```

| Name | Description | Default | Control |
| --- | --- | --- | --- |
| theme | Altera segmento dos estilos string |  | Choose option... brad-theme-classic brad-theme-corporate brad-theme-empresas brad-theme-exclusive brad-theme-prime brad-theme-private brad-theme-next brad-theme-expresso brad-theme-classic-old brad-theme-exclusive-old brad-theme-prime-old brad-theme-private-old brad-theme-agora brad-theme-afluentes |
| elevation | string | - | Choose option... brad-shadow-0 brad-shadow-10 brad-shadow-20 brad-shadow-30 brad-shadow-40 brad-shadow-top-10 brad-shadow-top-20 brad-shadow-top-30 brad-shadow-top-40 brad-shadow-left-10 brad-shadow-left-20 brad-shadow-left-30 brad-shadow-left-40 brad-shadow-right-10 brad-shadow-right-20 brad-shadow-right-30 brad-shadow-right-40 |
