# PreventSelect

A propriedade .brad-prevent-select pode ser usada para desabilitar a seleção em uma página inteira ou apenas em elementos específicos. .

# Uso do HTML
Prevent Select
```
<p class="brad-prevent-select">This text cannot be selected.</p>
Copy
```

Para habilitar a seleção de elementos específicos, é só usar:

```
<p class="brad-allow-select">This text can be selected.</p>
Copy
```
Exemplo
```
<p class="brad-allow-select">This text can be selected.</p>
<p class="brad-prevent-select">This text cannot be selected.</p>
```
