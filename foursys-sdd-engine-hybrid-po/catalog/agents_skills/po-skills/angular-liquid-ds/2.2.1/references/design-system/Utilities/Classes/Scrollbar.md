# Scrollbar

A scrollbar (vertical e horizontal) pode ser customizada utilizando a classe .brad-scrollbar.
Ela irá aparecer somente quando houver interação do usuário com o elemento que possui rolagem, permanecendo em seu estado default e alterando para os estados de hover e pressed caso a interação seja direta com a scrollbar.
Obs.: Devido a limitações do browser Mozilla Firefox, as customições da scrollbar não serão aplicadas, seguindo o layout default do browser.
