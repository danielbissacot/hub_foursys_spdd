# Foursys SDD Engine

O Maestro da Engenharia de Software Foursys. Este plugin implementa a metodologia **Spec-Driven Development (SDD)** integrada ao AI Governance Hub.

## 🚀 Funcionalidades
- **Constitution**: Governança técnica e regras de ouro.
- **Specify**: Refinamento de User Stories.
- **Plan**: Desenho técnico arquitetural.
- **Tasks**: Decomposição atômica de tarefas.
- **Implement**: Codificação assistida por Agentes Especialistas.

## 🛠️ Como usar
Use os comandos `/foursys.*` no chat ou utilize a barra lateral **Foursys SDD**.
