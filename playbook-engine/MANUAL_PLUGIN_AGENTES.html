GUIA DE INICIALIZAÇÃO

Potencialize seu código com IA

O Agentes Foursys é o seu co-piloto definitivo, integrando playbooks de engenharia, governança e automação diretamente no seu VS Code.

Como Começar

📥

Instalação

Arraste o arquivo agentes-foursys-1.0.0.vsix para dentro do VS Code ou use o comando "Install from VSIX".

⚡

Conexão

Na barra lateral, clique em CONECTAR AO HUB. Isso baixará os Agentes e Playbooks necessários para o seu projeto.

📁

doc_projeto

O plugin criará automaticamente uma pasta doc_projeto/. Todos os artefatos de IA serão salvos lá.

O Ciclo de Vida (Lifecycle)

01

Inserir História

Inicie seu projeto gerando o template oficial de User Story. Preencha a narrativa e a Tecnologia (Angular, Java ou COBOL).

02

Refinamento & Desenho

Use os botões 2 e 3 para validar requisitos e gerar o Plano de Implementação Técnica. A IA analisa riscos e define a arquitetura.

03

Desenvolver (O Motor)

O botão de ouro. A IA gera o código baseado no plano técnico e o plugin extrai e cria os arquivos fisicamente no seu workspace.

04

Qualidade & Testes

Realize Code Review e gere Testes Unitários automaticamente. O plugin identifica as classes e cria os arquivos .spec ou Test.java.

Comandos de Chat

Chame o @agentes_foursys no chat do Copilot para comandos rápidos:

/refinar

Refina a User Story atual.

/desenvolver

Gera o código no chat e no disco.

/review_angular

Code review específico para Angular.

/teste_java

Gera testes para Spring Boot.

Dica de Ouro 💡

Sempre mantenha o campo Tecnologia: preenchido na sua User Story. É através dele que o plugin sabe qual Agente Especialista deve chamar para te ajudar.
